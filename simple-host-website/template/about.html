About This Template

Designed to look real without adding complexity.

Students can inspect these files to see a straightforward static site structure: separate pages, a shared stylesheet, one script, and a local image asset.

What to Notice

  • Navigation works with relative links only.
  • Shared assets live in `css/`, `js/`, and `images/` folders.
  • The layout adapts cleanly from mobile to desktop.
  • Comments explain the purpose of each main section.

Why It Deploys Easily

There is no framework, package manager, or build step. A static host only needs these files as-is, which makes this a reliable reference project for training.
