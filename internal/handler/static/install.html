Get started

Ship a website by
talking to your AI.

No git, no dashboard, no signup form. There are two ways in, depending on where your AI lives: install the skill if you're in a coding agent, or just paste a link if you're in a web chat like ChatGPT.

For coding agents · the skill

Up and running in three steps.

Works with Claude Code, Codex CLI, opencode, and Cursor — anything with a shell.

  1. 1

    Install the skill

    One command drops it into your agent's skills folder:

    curl -fsSL https://simple-host.app/install.sh | sh

    Codex CLI → append -s -- ~/.agents/skills. opencode / Cursor → point it at their skills dir. view install.sh

  2. 2

    Describe what you want

    In a fresh chat, point at your files or just describe the site: "Deploy this folder to Simple Host," or "Build me a waitlist page with email signups."

  3. 3

    It ships — then iterate

    Live at https://your-name.simple-host.app in under a minute. Change it by chatting: "make the hero green," "add a live counter."

example prompt 
# Step 1 — install the skill (once)
curl -fsSL https://simple-host.app/install.sh | sh

# Step 2 — describe your site
Build a coming-soon page for my app
"PlantPal" with an email signup list
and a live "join N others" counter.

# Step 3 — ship it
Use the simple-host skill to deploy it.

# iterate, in plain English
"make the hero forest-green"
"add a countdown to launch day"
For web AI · just a link

No install? Paste a link.

Any browser assistant that can read a web page — ChatGPT, Gemini, Claude, Copilot. Give it the link, describe what you want, and read it the code it emails you.

Give your AI this https://simple-host.app/llms.txt
  1. 1

    Paste the link & describe it

    Drop the /llms.txt link into the chat and say what you want built. The assistant reads it and learns the whole API itself.

  2. 2

    Read it the 6-digit code

    It signs you in by email — you just paste back the code from your inbox. First deploy creates your account; no form to fill.

  3. 3

    Get your live URL

    It builds and deploys, then hands you https://your-name.simple-host.app. Ask for changes in plain English.

chatgpt · gemini · copilot 
You
Build me a wedding RSVP page and put it
online. Instructions: https://simple-host.app/llms.txt
My email is priya@example.com

Assistant
✓ Read the instructions at /llms.txt
✓ Emailed a 6-digit code — paste it here
  → you: 481920
✓ Built the page & deployed it
→ https://priya-wedding.simple-host.app

The only requirement is that the assistant can make web requests. If one genuinely can't, switch to one that can — everything else is automatic.
